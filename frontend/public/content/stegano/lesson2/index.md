# Файлы

Каждый файл имеет формат - структуру данных, которая оперделяет, как его нужно читать и интерпретировать. Даже если у двух файлов одинаковое расширение, их содрежимое может кардинально различаться.

Существую основные типы файлов:
- __Текстовые__ - содержат символы (.txt, .docx, .md)
- __Изображения__ - хранят пиксели (.jpg, .png, .bmp)
- __Аудио__ - набор звуковых данных (.mp3, .wav)
- __Видео__ - последовательность кадров (.mp4, .avi, .mov)

Чтобы понять какой перед вами файл, используют иинструмент - команду __file__
```
file - это утилита, которая анализирует содержимое файла. Она смотрит на сигнатуры и магические байты и определяет настоящий тип данных.
```
Пример использования: `file example.png`
Она вернет что-то похожее: `example.png: PNG image data, 1024 x 1024, 8-bit/color RGB`


Откройте [Фотографию](image1.png), которую мы вам оставили и прочитайте текст.

Чтобы подменить файл, можно просто поменять расширение, поэтому чтобы открыть файл, переименуйте его и верните нужное расширение.
Однако не всегда это помогает, потому что это слишком просто, чаще всего меняют ___сигнатуру___ файла.

Каждый файл имеет сигнатуру - уникальный набор характеристик, по которым можно определить его настоящий формат.
Сигнатура включает:
1. Первый байты файла (магические байты)
    - главная часть сигнатуры, уникальный ___паспорт___ формата.
2. Структуру заголовка
    - некоторые форматы содержат служебную информацию, размерность, версии и другие данные.
3. Смещение сигнатуры
    - в редких случаях характерные байты находяться не в начале файла (например, у некоторых аудиоформатов)

Посмотреть полный список сигнатур можно ___[здесь](https://en.wikipedia.org/wiki/List_of_file_signatures)___.
